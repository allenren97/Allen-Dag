# Obsidian vault — ETL repo docs

## Open in Obsidian

1. Install [Obsidian](https://obsidian.md/).
2. **Open folder as vault** → choose this directory:  
   `docs/obsidian/`  
   (full path on your machine: inside the git repo under `docs/obsidian`).

## Notes

- Notes use **wikilinks** like `[[03 - create_dag]]` — Obsidian resolves them to the matching `.md` file.
- Optional: enable **Mermaid** in Settings → **Reading** (or use a community plugin) to render the diagram in `01 - Flow from Excel to Blob.md`.

## Start reading

Open **`00 - Index.md`** — it is the map of content. Narrative notes **`00`–`10`** (plus this `README`) describe architecture; **`code/*.md`** embeds matching Python for deep dives.

## `code/` folder (embedded Python)

Markdown files under **`code/`** are generated by **`sync_code_snippets.py`** (full copies of selected `common_lib` modules). Regenerate after editing Python:

```bash
python3 docs/obsidian/sync_code_snippets.py
```

Wikilinks: use `[[code/write_dag_file]]` style from notes in this vault root (see [[00 - Index]] for the full table).

## Maintenance

Update narrative notes when behavior changes. Refresh **`code/*.md`** with the sync script above; update `sync_code_snippets.py` if you add new files to the manifest.
